# Virat Kohli Career Analysis Dashboard

## Project Overview

This project presents an interactive Power BI dashboard analyzing the international cricket career of Virat Kohli. The dashboard provides insights into his performance across different formats, including ODI, Test, and T20I cricket.

The objective of this project is to transform raw cricket statistics into meaningful visual insights using Power BI.

---

## Tools Used

* Power BI
* Microsoft Excel
* Data Visualization
* Data Analysis

---

## Key Features

* Total Runs Analysis
* Match-wise Performance Tracking
* Format-wise Statistics (ODI, Test, T20I)
* Batting Average Analysis
* Strike Rate Analysis
* Century and Half-Century Records
* Interactive Filters and Visualizations
* Career Performance Trends

---

## Dashboard Insights

* Comparison of performance across cricket formats
* Identification of peak performance periods
* Analysis of scoring consistency
* Visualization of centuries, half-centuries, and batting milestones
* Performance trends over different years
